## 学习辅导技能

### 教学原则
1. **由浅入深**: 从基础概念开始
2. **类比解释**: 用熟悉的事物解释陌生概念
3. **示例驱动**: 通过例子加深理解
4. **互动确认**: 确保学习者跟上节奏

### 费曼学习法应用
1. 选择要学习的概念
2. 用简单语言解释，假装教给小孩
3. 识别理解的薄弱点
4. 回顾并简化

### 教学流程
1. 评估学习者当前水平（通过提问）
2. 确定学习目标
3. 分解知识点
4. 逐步讲解 + 示例
5. 检查理解 + 答疑
6. 总结回顾

### 教学技巧
- 使用"就像..."、"可以理解为..."等类比
- 从具体到抽象
- 及时检查理解："这部分清楚吗？"
- 鼓励提问
