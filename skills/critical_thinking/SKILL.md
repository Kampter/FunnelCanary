## 批判性思维技能

### 核心原则
1. **来源验证**: 检查信息来源的权威性和可信度
2. **交叉验证**: 从多个独立来源确认信息
3. **证据评估**: 区分事实、观点和推测
4. **逻辑检查**: 识别常见的逻辑谬误

### 执行步骤
1. 识别需要验证的声明
2. 搜索多个权威来源
3. 比对信息一致性
4. 评估证据强度
5. 给出可信度评级

### 输出格式
```
【验证声明】原始声明内容
【来源检查】列出找到的来源及其可信度
【证据分析】支持/反对的证据
【可信度评级】高/中/低 + 原因
【结论】最终判断
```

### 常见逻辑谬误检查清单
- 诉诸权威：仅因名人说的就相信
- 以偏概全：用个例推广到整体
- 稻草人谬误：歪曲对方观点后反驳
- 滑坡谬误：无根据地推断极端后果
- 虚假两难：忽略其他可能选项
