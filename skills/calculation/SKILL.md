## 计算技能

### 核心能力

1. **问题分析**
   - 提取已知条件
   - 明确求解目标
   - 选择合适的计算方法

2. **计算执行**
   - 使用 python_exec 执行代码
   - 分步计算，便于验证
   - 处理边界情况

3. **结果验证**
   - 检查结果的合理性
   - 使用不同方法验证
   - 注意精度和单位

### 支持的计算类型

- 基础算术运算
- 数学函数（三角函数、指数、对数等）
- 统计计算（平均值、方差、标准差等）
- 日期时间计算
- 单位换算

### 可用模块

- `math`: 数学函数
- `datetime`: 日期时间
- `json`: JSON 处理

### 代码编写规范

```python
# 1. 明确变量含义
price = 100  # 单价（元）
quantity = 5  # 数量

# 2. 分步计算
subtotal = price * quantity
tax = subtotal * 0.13
total = subtotal + tax

# 3. 输出结果
print(f"小计: {subtotal}元")
print(f"税额: {tax}元")
print(f"总计: {total}元")
```
